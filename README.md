### What is Cisco RADKit?
📡  **Cisco RADKit** (Remote Automation Development Kit) is a Software Development Kit (SDK): a set of ready-to-use tools and Python modules that enable efficient and scalable interactions with local or remote network equipment.

RADKit allows you to:

- Interactively connect to remote terminals, Web UIs, or desktops
- Use a powerful and approachable API for network automations and data collection
- Support protocols such as SSH, NETCONF, OpenAPI/Swagger, and others
- Securely invite Cisco support engineers to access only the data you choose to share

RADKit is available at no additional cost with valid Cisco Support Contracts.

> Official docs are available [**here!**](https://radkit.cisco.com/docs/introduction.html)
